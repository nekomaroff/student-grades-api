# Student Grades API

Реализован REST API для загрузки и анализа оценок студентов.

## Описание решения

В рамках задания выполнено:
- Создан HTTP-сервис на FastAPI
- Реализована загрузка оценок студентов из CSV-файла в базу данных PostgreSQL
- Реализованы запросы для получения статистики по количеству оценок «2» у студентов
- Данные хранятся в реляционной базе PostgreSQL
- Структура базы данных создаётся с помощью SQL-скрипта
- База данных разворачивается с использованием Docker Compose
- Для взаимодействия с БД используется psycopg2

## Запуск проекта

1. Запуск базы данных PostgreSQL: docker-compose up -d
2. Установка зависимостей: pip install -r requirements.txt
3. Запуск сервера: uvicorn app.main:app --reload
4. После запуска API доступно по адресу: http://127.0.0.1:8000 (Swagger UI: http://127.0.0.1:8000/docs)
## SQL-скрипт

SQL-скрипт создания таблицы находится в файле sql/init.sql

## Реализованные API-методы

- POST /upload-grades - загрузка CSV-файла с оценками студентов
- GET /students/more-than-3-twos - студенты с более чем 3 оценками "2"
- GET /students/less-than-5-twos - студенты с менее чем 5 оценками "2"
